Disaster Clippy - Embed Widget

How to Use

This is a simple chat widget that connects to the Disaster Clippy AI. External websites can embed this widget to provide their users with access to DIY, survival, and humanitarian knowledge.

API Endpoint

POST to /api/v1/chat

Request Format

{
    "message": "How do I purify water?",
    "session_id": "optional-session-id"
}

Response Format

{
    "response": "Here are several methods to purify water...",
    "session_id": "2025-01-15T10:30:00"
}

Quick Embed

Visit /api/v1/embed to get ready-to-use HTML code for your website.
