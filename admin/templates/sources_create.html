Scrape from a live source

MediaWiki site

Any site running MediaWiki — Appropedia, Fandom wikis, or your own instance.

python -m cli.ingest scrape mediawiki --url https://www.appropedia.org

Appropedia (preset)

Pre-configured for Appropedia.org with recommended category filters.

python -m cli.ingest scrape appropedia

Fandom wiki (preset)

Any Fandom wiki — pass the subdomain.

python -m cli.ingest scrape fandom --wiki solarcooking

Static HTML site

Crawls a site via sitemap.xml. Supply a CSS selector to target the main content.

python -m cli.ingest scrape static --url https://example.com --selector article

Substack newsletter

Export your subscriber list from Substack, then pass the CSV and the newsletter URL.

python -m cli.ingest scrape substack --csv posts.csv --url https://yourname.substack.com

Import from a file or archive

Single PDF

Process one PDF file and add it to the vector database.

python -m cli.ingest scrape pdf --file /path/to/document.pdf

PDF folder

Batch-process all PDFs in a folder. Use --sectioned for structured docs like building codes.

python -m cli.ingest scrape pdf --folder /path/to/pdfs --sectioned

PDF collection (managed)

Organise PDFs into named collections with licensing and topic tags before indexing.

python -m cli.ingest pdf create my-collection python -m cli.ingest pdf import inbox-item python -m cli.ingest pdf process my-collection

Full pipeline (scrape, index, and make searchable)

Run these three steps in order for any new source. Replace the scrape command with whichever type applies.

1. Scrape

python -m cli.ingest scrape mediawiki --url https://www.appropedia.org

2. Index into vector database

python -m cli.ingest sync

3. Check the result

python -m cli.ingest stats
